JavaScript

まず、メモ帳機能を使ってブラウザに文字を表示してみよう！

メモ帳を開いて、下のコードをコピー、ペーストします。改行や空白もそのままコピーしてください。 

          
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>JavaScriptをためしてみよう</title>
</head>
<body>
<h1>
クリックしてみよう!
</h1>
<script src="main.js"> </script> 
</body>
</html>

デスクトップ上にファイルを保存します。保存の仕方はHTML+CSSを参照してください。
保存したファイルをブラウザで開くと文字が表示されます。

クリック操作で文字の色や大きさを変更できるようにしよう。

新しくメモ帳を開いて下のコードをコピー、ペーストして同じようにデスクトップに保存してください。必ずデスクトップに保存し、ファイル名は　main.js　にしてください。

          
const h1 = document.querySelector('h1');
document.addEventListener('click', () => {
  h1.textContent = 'テキストの色が変わりました！';
  h1.style.color = 'red';
  h1.style.fontSize = '16px';
});

先ほどのブラウザをリロード refresh してください。文字をクリックすると表示が変わりましたね。今回は最初の文字に戻す設定はしていませんので、最初の文字に戻したいときはページをリロードしてください。

メモ帳でmain.jsファイルをひらいて、textContentには表示させたい文章、colorは文字の色、fontSizeは文字の大きさを変更し、ファイルを上書き保存します。

ブラウザでのファイルをリロードして文字をクリックするとクリックしたときの表示される内容が変わります。
